Getting started is easy.

Install nodejs version 0.8.0 or later.

Install phantomjs version 1.6.0

git clone https://github.com/venusjs/venus.git

cd venus && npm install

Add venus/bin to your path

That's it! To verify everything is working, try this...

venus run --test venus/examples/01-SimpleMochaTest/specs/Greeter.spec.js --phantom
